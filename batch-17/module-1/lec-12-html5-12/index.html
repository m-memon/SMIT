HTML5 Video & Media Embedding

• Every element have any by default CSS property in HTML5:

  • English Description:
    In HTML5, every element comes with default CSS properties assigned by the browser. These properties include things like font size, color, margins, padding, etc. Although these styles can be overridden with custom CSS, the default styles determine the initial look of elements on a webpage.
  • Roman Urdu Description:
    HTML5 mein har element ke saath browser se default CSS properties hoti hain, jisme font size, color, margins, padding waghera shamil hoti hain. Yeh properties customize CSS se replace ki ja sakti hain, lekin default styles webpage ke elements ka initial look tay karte hain.
  • Example Code:
    <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Default CSS Properties Example</title>
    </head>
    <body>
        <p>This is a paragraph with default styles.</p>
    </body>
</html>

• <favicon> element tag of HTML5:

  • English Description:
    The <favicon> tag is used to define a shortcut icon for the webpage. It is typically displayed in the browser tab or bookmark bar. However, the <favicon> tag itself is not part of HTML5. Instead, the link to the favicon is added using a <link> tag.
  • Roman Urdu Description:
    <favicon> tag ka use webpage ke liye shortcut icon define karne ke liye hota hai. Yeh icon browser tab ya bookmark bar mein dikhai deta hai. Lekin, <favicon> tag HTML5 ka hissa nahi hai, isliye isko <link> tag ke zariye add kiya jata hai.
  • Example Code:
    <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="icon" href="favicon.ico" type="image/x-icon">
        <title>Favicon Example</title>
    </head>
    <body>
        <h1>Check the tab for the favicon!</h1>
    </body>
</html>

• <iframe> element tag of HTML5:

  • English Description:
    The <iframe> tag is used to embed another HTML document inside the current webpage. It allows you to display content such as videos, maps, or other web pages without leaving the current page.
  • Roman Urdu Description:
    <iframe> tag ka use kisi aur HTML document ko apne webpage mein embed karne ke liye hota hai. Isse aap videos, maps, ya doosri web pages ko bina page chhode display kar sakte hain.
  • Example Code:
    <iframe src="https://www.example.com" width="600" height="400"></iframe>

• What is embed link:

  • English Description:
    An embed link is a URL that allows embedding content, such as videos, maps, or other interactive media, directly into a webpage. It is typically used with tags like <iframe>, <embed>, or <object>.
  • Roman Urdu Description:
    Embed link ek URL hota hai jo kisi content ko direct webpage mein embed karne ke liye use hota hai, jaise videos, maps, ya interactive media. Yeh zyada tar <iframe>, <embed>, ya <object> tags ke sath use hota hai.
  • Example Code:
    <iframe src="https://www.youtube.com/embed/example" width="560" height="315"></iframe>

• <video> element tag in HTML5:

  • English Description:
    The <video> element in HTML5 is used to embed video content directly into a webpage. It supports multiple video formats and allows users to control playback with attributes like play, pause, volume, and fullscreen.
  • Roman Urdu Description:
    HTML5 ka <video> element webpage mein video content embed karne ke liye use hota hai. Yeh kai video formats ko support karta hai aur users ko playback control karne ke liye play, pause, volume, aur fullscreen jaise attributes provide karta hai.
  • Example Code:
    <video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

• <source> element tag of <video> element tag in HTML5:

  • English Description:
    The <source> element is used inside the <video> tag to specify multiple video files for the browser to choose from. It allows you to offer different video formats to ensure compatibility across browsers.
  • Roman Urdu Description:
    <source> element <video> tag ke andar use hota hai jisme aap multiple video files specify kar sakte hain taake browser apne compatible format ko choose kar sake.
  • Example Code:
    <video controls>
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.ogv" type="video/ogg">
    Your browser does not support the video tag.
</video>

• src attribute of <source> element tag of <video> element tag in HTML5:

  • English Description:
    The src attribute in the <source> element specifies the path to the video file. This tells the browser where to find the media to be played.
  • Roman Urdu Description:
    src attribute <source> element mein video file ka path specify karta hai, jo browser ko yeh batata hai ke media kahan se play karna hai.
  • Example Code:
    <video controls>
    <source src="movie.mp4" type="video/mp4">
</video>

• controls attribute of <video> element tag in HTML5:

  • English Description:
    The controls attribute is added to the <video> element, not the <source> element. It provides the user with playback controls, such as play, pause, volume, and fullscreen options.
  • Roman Urdu Description:
    controls attribute <video> element ke saath use hota hai, <source> element mein nahi. Yeh user ko playback controls provide karta hai, jaise play, pause, volume, aur fullscreen options.
  • Example Code:
    <video controls>
    <source src="movie.mp4" type="video/mp4">
</video>

• muted attribute of <video> element tag in HTML5:

  • English Description:
    The muted attribute in the <video> element silences the audio of the video. It is often used for autoplaying videos where sound is not needed.
  • Roman Urdu Description:
    muted attribute <video> element mein use hota hai jo video ka audio mute kar deta hai. Yeh zyada tar autoplay videos mein use hota hai jahan awaaz ki zaroorat nahi hoti.
  • Example Code:
    <video muted autoplay>
    <source src="movie.mp4" type="video/mp4">
</video>

• loop attribute of <video> element tag in HTML5:

  • English Description:
    The loop attribute causes the video to automatically restart once it ends, providing continuous playback.
  • Roman Urdu Description:
    loop attribute video ko automatically restart karne ke liye hota hai jab video khatam ho jata hai, taake continuous playback ho sake.
  • Example Code:
    <video loop>
    <source src="movie.mp4" type="video/mp4">
</video>

• poster attribute of <video> element tag in HTML5:

  • English Description:
    The poster attribute specifies an image to be shown before the video starts playing. This image appears as a placeholder while the video is loading.
  • Roman Urdu Description:
    poster attribute ek image specify karta hai jo video ke play hone se pehle dikhayi deti hai. Yeh image video load hone tak placeholder ki tarah kaam karti hai.
  • Example Code:
    <video poster="poster-image.jpg">
    <source src="movie.mp4" type="video/mp4">
</video>
