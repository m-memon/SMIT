HTML5 Form Attributes

• <textarea> element tag of <form> element tag in HTML5:

  • English Description:
    The <textarea> element is used to create a multi-line text input field within a <form> in HTML5. It allows users to enter large amounts of text, such as comments or feedback.
  • Roman Urdu Description:
    <textarea> element ko HTML5 mein <form> ke andar multi-line text input field banane ke liye use kiya jata hai. Is se users ko zyada text daalne ka mauka milta hai, jaise comments ya feedback.
  • Example Code:
    <form>
    <label for="feedback">Feedback:</label>
    <textarea id="feedback" name="feedback" rows="4" cols="50"></textarea>
</form>

• cols attribute of <textarea> element tag of <form> element tag in HTML5:

  • English Description:
    The cols attribute specifies the visible width of a <textarea> element in terms of character columns. It controls how many characters can fit in one line.
  • Roman Urdu Description:
    cols attribute <textarea> element ki visible width ko character columns ke hisaab se specify karta hai. Yeh decide karta hai ke ek line mein kitne characters fit ho sakte hain.
  • Example Code:
    <textarea cols="50" rows="4"></textarea>

• rows attribute of <textarea> element tag of <form> element tag in HTML5:

  • English Description:
    The rows attribute defines the visible height of the <textarea> element by specifying how many lines of text it should display.
  • Roman Urdu Description:
    rows attribute <textarea> element ki visible height ko define karta hai, jisme yeh specify kiya jata hai ke kitni lines of text display honi chahiye.
  • Example Code:
    <textarea rows="6" cols="50"></textarea>

• hidden attribute of <input /> element tag of <form> element tag in HTML5:

  • English Description:
    The hidden attribute can be added to an <input /> element to hide it from the user interface. This makes the input element not visible on the page, but it still exists in the form.
  • Roman Urdu Description:
    hidden attribute <input /> element mein add karke usko user interface se hide kiya ja sakta hai. Is se input element page par nazar nahi aata, lekin form mein still exist karta hai.
  • Example Code:
    <form>
    <input type="hidden" name="user_id" value="12345">
</form>

• disabled attribute of <option> element tag of <select> element tag of <form> element tag in HTML5:

  • English Description:
    The disabled attribute in the <option> element of a <select> dropdown disables the option, making it unselectable by the user.
  • Roman Urdu Description:
    disabled attribute <select> dropdown ke <option> element mein option ko disable kar deta hai, jisse user usko select nahi kar sakta.
  • Example Code:
    <select>
    <option value="1">Option 1</option>
    <option value="2" disabled>Option 2 (Disabled)</option>
    <option value="3">Option 3</option>
</select>

• selected attribute of <option> element tag of <select> element tag of <form> element tag in HTML5:

  • English Description:
    The selected attribute is used to pre-select an option in a dropdown menu within a <select> element.
  • Roman Urdu Description:
    selected attribute <select> dropdown menu mein kisi option ko pehle se select karne ke liye use hota hai.
  • Example Code:
    <select>
    <option value="1">Option 1</option>
    <option value="2" selected>Option 2 (Selected)</option>
    <option value="3">Option 3</option>
</select>

• multiple attribute of <select> element tag of <form> element tag in HTML5:

  • English Description:

  • Roman Urdu Description:

• How to comment in HTML5:

  • English Description:
    In HTML5, comments are written between <!-- and --> tags. Anything between these tags is ignored by the browser and does not appear on the webpage.
  • Roman Urdu Description:
    HTML5 mein comments <!-- aur --> tags ke beech likhe jate hain. In tags ke beech ka content browser ignore karta hai aur webpage par nahi dikhai deta.
  • Example Code:
    <!-- This is a comment -->
<p>This is a visible text.</p>
<!-- This paragraph is commented out -->
