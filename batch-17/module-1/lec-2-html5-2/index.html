HTML5 Attributes Explained

• Attribute in HTML5:

  • English Description:
    An attribute in HTML5 is a part of an element that provides additional information about that element. Attributes are placed inside the opening tag.
  • Roman Urdu Description:
    HTML5 mein attribute ek element ka hissa hota hai jo us element ke baare mein zyada maloomat deta hai. Attributes opening tag ke andar likhe jate hain.
  • Example Code:
    <a href="https://example.com">Click Here</a>

• Types Of Attributes in HTML5:

  • English Description:
    HTML5 supports various types of attributes, such as global attributes, event attributes, and specific attributes for individual elements.
  • Roman Urdu Description:
    HTML5 mein mukhtalif types ke attributes hote hain, jaise ke global attributes, event attributes, aur individual elements ke liye specific attributes.
  • Example Code:
    <img src="image.jpg" alt="Image Description" />

• Global attributes in HTML5:

  • English Description:
    Global attributes are attributes that can be used on any HTML element. Examples include id, class, style, etc.
  • Roman Urdu Description:
    Global attributes wo attributes hain jo har HTML element par istemal kiye ja sakte hain. Misal ke taur par id, class, style, etc.
  • Example Code:
    <div id="header" class="main-header" style="background-color: blue;">
  <p>Welcome to the site!</p>
</div>

• Event attributes in HTML5:

  • English Description:
    Event attributes allow you to assign an event handler to an HTML element. They are used to run a JavaScript function when a specific event occurs.
  • Roman Urdu Description:
    Event attributes aapko HTML element par event handler assign karne ki sahulat dete hain. Yeh tab istemal hote hain jab koi khaas event hota hai aur JavaScript function run hota hai.
  • Example Code:
    <button onclick="alert('Hello, World!')">Click Me</button>

• Specific attributes for individual elements in HTML5:

  • English Description:
    Some attributes are specific to certain HTML elements. For example, the href attribute is specific to anchor (<a>) tags, and src is specific to <img> tags.
  • Roman Urdu Description:
    Kuch attributes kuch khaas HTML elements ke liye hotay hain. Misal ke taur par href attribute <a> tag ke liye aur src attribute <img> tag ke liye hota hai.
  • Example Code:
    <a href="https://example.com">Visit Example</a>  
<img src="image.jpg" alt="Sample Image">

• lang attribute of <html> element tag in HTML5:

  • English Description:
    The lang attribute specifies the language of the content inside the <html> element. It helps search engines and browsers identify the language.
  • Roman Urdu Description:
    lang attribute <html> element ke andar content ki zaban specify karta hai. Yeh search engines aur browsers ko zaban ka pata chalne mein madad karta hai.
  • Example Code:
    <html lang="en">

• <b> element tag in HTML5:

  • English Description:
    The <b> element is used to make text bold, but it doesn’t convey any special importance or emphasis.
  • Roman Urdu Description:
    <b> element ka istemal text ko bold karne ke liye hota hai, lekin yeh koi khaas ahmiyat ya emphasis nahi deta.
  • Example Code:
    <b>This is bold text</b>

• <i> element tag in HTML5:

  • English Description:
    The <i> element is used to italicize text, typically for emphasizing certain words or phrases.
  • Roman Urdu Description:
    <i> element ka istemal text ko italic banane ke liye hota hai, jo aam tor par kisi lafz ya jumlay ko emphasize karne ke liye hota hai.
  • Example Code:
    <i>This is italic text</i>
