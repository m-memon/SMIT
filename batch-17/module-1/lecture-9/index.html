HTML5 Table Attributes

• width attribute in <table> element tag of HTML5:

  • English Description:
    Defines the width of the table in pixels or percentage of the containing element.
  • Roman Urdu Description:
    Table ki choraai ko pixels ya percentage mein set karta hai.
  • Example Code:
    <table width="100%">
    <tr><td>Full Width Table</td></tr>
</table>

• height attribute in <table> element tag of HTML5:

  • English Description:
    Specifies the height of the table. However, it is not a standard HTML5 attribute and is better controlled with CSS.
  • Roman Urdu Description:
    Table ki lambai set karta hai, lekin HTML5 mein ye standard nahi hai. CSS se use karna behtar hota hai.
  • Example Code:
    <table style="height:200px;">
    <tr><td>Fixed Height Table</td></tr>
</table>

• bgcolor attribute of <tr> element tag in <table> element tag in HTML5:

  • English Description:
    Sets the background color of a table row. This attribute is obsolete in HTML5; use CSS instead.
  • Roman Urdu Description:
    Table row ka background color set karta hai. HTML5 mein ye purana tareeqa hai, CSS use karein.
  • Example Code:
    <table>
    <tr bgcolor="#f2f2f2">
        <td>Colored Row</td>
    </tr>
</table>

• bgcolor attribute of <td> element tag in <tr> element tag of <table> element tag in HTML5:

  • English Description:
    Sets the background color of a specific table cell. This attribute is obsolete in HTML5; prefer using CSS.
  • Roman Urdu Description:
    Table cell ka background color set karta hai. HTML5 mein ye purani attribute hai, CSS istemal karein.
  • Example Code:
    <table>
    <tr>
        <td bgcolor="#dcdde1">Colored Cell</td>
    </tr>
</table>

• align attribute of <td> element tag in <tr> element tag of <table> element tag in HTML5:

  • English Description:
    Specifies horizontal alignment of cell content. This is deprecated in HTML5; use CSS instead.
  • Roman Urdu Description:
    Cell ke andar text ya content ka alignment set karta hai (left, center, right). HTML5 mein ye purana tareeqa hai.
  • Example Code:
    <table>
    <tr>
        <td align="center">Centered Text</td>
    </tr>
</table>

• colspan attribute of <td> element tag in <tr> element tag of <table> element tag in HTML5:

  • English Description:
    Allows a single table cell to span across multiple columns.
  • Roman Urdu Description:
    Ek cell ko kai columns tak failane ke liye use hoti hai.
  • Example Code:
    <table border="1">
    <tr>
        <td colspan="2">Spanning Two Columns</td>
    </tr>
    <tr>
        <td>Column 1</td>
        <td>Column 2</td>
    </tr>
</table>

• rowspan attribute of <td> element tag in <tr> element tag of <table> element tag in HTML5:

  • English Description:
    Allows a single table cell to span across multiple rows.
  • Roman Urdu Description:
    Ek cell ko kai rows tak failane ke liye use hoti hai.
  • Example Code:
    <table border="1">
    <tr>
        <td rowspan="2">Spanning Two Rows</td>
        <td>Row 1, Col 2</td>
    </tr>
    <tr>
        <td>Row 2, Col 2</td>
    </tr>
</table>
