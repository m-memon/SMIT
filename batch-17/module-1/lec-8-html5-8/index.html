HTML5 Table Elements

• Align attribute of <img> element tag in HTML5:

  • English Description:
    The align attribute was used in older HTML versions to control the alignment of images on a webpage. However, in HTML5, it’s deprecated, and CSS should be used for image alignment instead.
  • Roman Urdu Description:
    align attribute purani HTML versions mein image ke alignment ko control karta tha. Lekin HTML5 mein yeh deprecated ho gaya hai aur ab image ki alignment CSS ke zariye ki jati hai.
  • Example Code:
    <!-- Deprecated in HTML5 -->
<img src="image.jpg" align="left">

• href Attributes of <a> element tag in HTML5:

  • English Description:
    The href attribute is used to specify the destination URL for the anchor (<a>) tag. This allows users to click the link and navigate to the given URL.
  • Roman Urdu Description:
    href attribute anchor (<a>) tag mein destination URL ko specify karta hai. Yeh user ko link par click karne par us URL par navigate karne ka moka deta hai.
  • Example Code:
    <a href="https://www.example.com">Click here</a>

• id selector attribute in HTML5:

  • English Description:
    The id attribute in HTML is used to assign a unique identifier to an HTML element. This identifier must be unique within the document, meaning no two elements should have the same id. This allows easy navigation and targeting of the element within the same page.
  • Roman Urdu Description:
    id attribute HTML mein kisi element ko unique identifier dene ke liye use hota hai. Yeh identifier page mein sirf ek hi element ke liye hona chahiye, matlab do elements ka same id nahi ho sakta. Yeh element ko easily target karne aur navigate karne mein madad karta hai.
  • Example Code:
    <p id="intro">Welcome to my website!</p>

• How to target id selector in HTML5:

  • English Description:
    In HTML, you can target an element using its id by creating a hyperlink (<a> tag) with an href attribute that refers to the id. To navigate to the element, simply use # followed by the id value in the href attribute.
  • Roman Urdu Description:
    HTML mein aap id ke zariye element ko target karne ke liye <a> tag ka use karte hain. href attribute mein id ke naam ke saath # symbol lagana hota hai. Is se aap us element tak navigate kar sakte hain.
  • Example Code:
    <!-- Link to navigate to the element with id="intro" -->
<a href="#intro">Go to Intro Section</a>

<!-- Target element -->
<p id="intro">This is the introduction section.</p>

• <table> element tag in HTML5:

  • English Description:
    The <table> element is used to represent tabular data. It consists of rows and columns, which are defined by <tr>, <td>, and other table-related elements.
  • Roman Urdu Description:
    <table> element tabular data ko represent karne ke liye use hota hai. Yeh rows aur columns par mabni hota hai jo ke <tr>, <td>, aur doosray table-related elements se define kiye jate hain.
  • Example Code:
    <table>
  <tr>
    <td>Row 1, Column 1</td>
    <td>Row 1, Column 2</td>
  </tr>
</table>

• <tbody> Element Tag of <table> Element Tag in HTML5:

  • English Description:
    The <tbody> tag is used to group the body content in a table. It is often used in combination with <thead> and <tfoot> for better structure and styling.
  • Roman Urdu Description:
    <tbody> tag table ke body content ko group karne ke liye use hota hai. Yeh aksar <thead> aur <tfoot> ke saath milkar table ki structure ko behtar banata hai.

  <thead>

      <th>Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Row 1, Column 1</td>
      <td>Row 1, Column 2</td>
    </tr>
  </tbody>
</table>
