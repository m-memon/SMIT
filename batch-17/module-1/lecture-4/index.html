HTML5 Layout Elements

• Inline elements in HTML5:

  • English Description:
    Inline elements do not start on a new line and only take up as much width as necessary.
  • Roman Urdu Description:
    Inline elements nayi line se start nahi hotay aur sirf jitni jagah zaroori ho utni hi lete hain.
  • Example Code:
    <span>This is inline text.</span>

• Block level elements in HTML5:

  • English Description:
    Block elements start on a new line and take up the full width available.
  • Roman Urdu Description:
    Block elements nayi line se shuru hotay hain aur poori available width le lete hain.
  • Example Code:
    <div>This is a block-level element.</div>

• <pre> element tag in HTML5:

  • English Description:
    The <pre> tag displays preformatted text with spaces and line breaks preserved.
  • Roman Urdu Description:
    <pre> tag aise text ko dikhata hai jisme spaces aur line breaks jese hain waise hi nazar aate hain.
  • Example Code:
    <pre>
Line 1
    Line 2
        Line 3
</pre>

• <u> element tag in HTML5:

  • English Description:
    The <u> tag underlines the text it contains.
  • Roman Urdu Description:
    <u> tag text ko underline karta hai.
  • Example Code:
    <p>
    This is
    <u>
        underlined
    </u> text.
</p>

• <ul> element tag in HTML5:

  • English Description:
    The <ul> tag defines an unordered (bulleted) list.
  • Roman Urdu Description:
    <ul> tag aik unordered ya bulleted list banata hai.
  • Example Code:
    <ul>
    <li>Apple</li>
    <li>Banana</li>
</ul>

• <ol> element tag in HTML5:

  • English Description:
    The <ol> tag defines an ordered (numbered) list.
  • Roman Urdu Description:
    <ol> tag aik ordered ya numbered list banata hai.
  • Example Code:
    <ol>
    <li>First</li>
    <li>Second</li>
</ol>

• <li> element tag of <ul> or <ol> element tag in HTML5:

  • English Description:
    The <li> tag defines a list item inside <ul> or <ol>.
  • Roman Urdu Description:
    <li> tag list ke items ko define karta hai <ul> ya <ol> ke andar.
  • Example Code:
    <ul>
    <li>Item 1</li>
</ul>

• type attribute in HTML5:

  • English Description:
    The type attribute defines the style of bullets or numbers in lists.
  • Roman Urdu Description:
    type attribute bullets ya numbers ke style ko set karta hai list mein.
  • Example Code:
    <ol type="A">
    <li>Option A</li>
    <li>Option B</li>
</ol>

• start attribute of <ol> element tag in HTML5:

  • English Description:
    The start attribute specifies the starting number of an ordered list.
  • Roman Urdu Description:
    start attribute ordered list ka starting number set karta hai.
  • Example Code:
    <ol start="5">
    <li>Five</li>
    <li>Six</li>
</ol>

• reversed attribute of <ol> element tag in HTML5:

  • English Description:
    The reversed attribute displays the ordered list in descending order.
  • Roman Urdu Description:
    reversed attribute ordered list ko ulta ya descending order mein dikhata hai.
  • Example Code:
    <ol reversed>
    <li>Three</li>
    <li>Two</li>
    <li>One</li>
</ol>

• <dl> element tag in HTML5:

  • English Description:
    The <dl> tag defines a description list of terms and definitions.
  • Roman Urdu Description:
    <dl> tag terms aur unki definitions ka list banata hai.
  • Example Code:
    <dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
</dl>

• <dt> element tag of <dl> element tag in HTML5:

  • English Description:
    The <dt> tag defines a term in a description list.
  • Roman Urdu Description:
    <dt> tag ek term ko define karta hai description list mein.
  • Example Code:
    <dt>CSS</dt>

• <dd> element tag of <dl> element tag in HTML5:

  • English Description:
    The <dd> tag describes the term listed by <dt>.
  • Roman Urdu Description:
    <dd> tag us term ki tafseel batata hai jo <dt> mein likhi hoti hai.
  • Example Code:
    <dd>Cascading Style Sheets</dd>
