• What is the full form of HTML?:

  • English Description:
    HTML stands for HyperText Markup Language, which is used to create webpages.
  • Roman Urdu Description:
    HTML ka pura naam HyperText Markup Language hai, jo webpages banane ke liye use hota hai.
  • Example Code:
    <!-- An example of HTML structure -->
<!DOCTYPE html>
<html>
    <head>
        <title>My First Webpage</title>
    </head>
    <body>
        <h1>Welcome to My Webpage</h1>
    </body>
</html>

• What is HyperText Markup Language?:

  • English Description:
    HTML is the standard language for creating webpages and web applications, structuring content with tags.
  • Roman Urdu Description:
    HTML webpages aur web applications banane ke liye standard zaban hai, jo content ko tags ke zariye structure karti hai.
  • Example Code:
    <!DOCTYPE html>
<html>
    <head>
        <title>HTML Example</title>
    </head>
    <body>
        <p>This is a paragraph in HTML.</p>
    </body>
</html>

• Folder Structure Explanation:

  • English Description:
    Folder structure refers to how files and directories are organized in a project, often with separate folders for HTML, CSS, JS, and images.
  • Roman Urdu Description:
    Folder structure se muraad hai ke kis tarah files aur directories ko project mein organize kiya jata hai, jisme HTML, CSS, JS aur images ke liye alag folders hote hain.
  • Example Code:
    /project
    /index.html
    /styles.css
    /script.js
    /images
        /logo.png

• What are Tags?:

  • English Description:
    Tags are special keywords in HTML used to define content elements, enclosed in angle brackets < >.
  • Roman Urdu Description:
    Tags HTML mein khas keywords hote hain jo content elements ko define karte hain, jo angle brackets < > mein likhe jate hain.
  • Example Code:
    <p>This is a paragraph tag.</p>

• What are Elements?:

  • English Description:
    An element is a combination of an opening tag, content, and a closing tag, like <p>This is a paragraph</p>.
  • Roman Urdu Description:
    Element ek combination hota hai opening tag, content, aur closing tag ka, jaise <p>This is a paragraph</p>.
  • Example Code:
    <p>This is a paragraph element.</p>

• Types of Tags:

  • English Description:
    Tags can be either block-level tags (e.g., <div>, <p>) or inline tags
    (e.g., <span>, <a>)
    b>.
  • Roman Urdu Description:
    Tags ya to block-level tags ho sakte hain (jaise <div>, <p>) ya inline tags (jaise <span>, <a>).
  • Example Code:
    <p>This is a block-level tag.</p>
<span>This is an inline tag.</span>

• What is an Opening Tag?:

  • English Description:
    An opening tag marks the start of an HTML element, such as <h1>.
  • Roman Urdu Description:
    Opening tag HTML element ke shuru hone ka indication deta hai, jaise <h1>.
  • Example Code:
    <h1>This is an opening tag.</h1>

• What is a Closing Tag?:

  • English Description:
    A closing tag indicates the end of an HTML element, like </h1>.
  • Roman Urdu Description:
    Closing tag HTML element ke end hone ka indication deta hai, jaise </h1>.
  • Example Code:
    <h1>This is a closing tag example.</h1>

• What is a Self-Closing Tag?:

  • English Description:
    Self-closing tags don’t require a closing counterpart, such as <img> or <br>.
  • Roman Urdu Description:
    Self-closing tags ko closing tag ki zaroorat nahi hoti, jaise <img> ya <br>.
  • Example Code:
    <img src="image.jpg" alt="Image">
<br>

• Difference Between Self-Closing Tag & Opening/Closing Tag:

  • English Description:
    A self-closing tag doesn’t have a separate closing tag, while opening/closing tags have both.
  • Roman Urdu Description:
    Self-closing tag ka apna closing tag nahi hota, jabke opening/closing tag mein dono hote hain.
  • Example Code:
    <!-- Self-closing tag -->
<img src="image.jpg" alt="Image">

• What is <html> Element Tag?:

  • English Description:
    The <html> tag defines the root of an HTML document, enclosing all other HTML elements.
  • Roman Urdu Description:
    <html> tag HTML document ka root define karta hai, jo baaki HTML elements ko enclose karta hai.
  • Example Code:
    <html>
    <head><title>Page Title</title></head>
    <body><h1>Hello World</h1></body>
</html>

• What is <head> Element Tag?:

  • English Description:
    The <head> tag contains meta-information about the document like title, character encoding, etc.
  • Roman Urdu Description:
    <head> tag document ki meta-information rakhta hai, jaise title, character encoding, waghera.
  • Example Code:
    <head>
    <title>My Webpage</title>
    <meta charset="UTF-8">
</head>

• What is <title> Element Tag?:

  • English Description:
    The <title> tag specifies the title of the webpage, shown in the browser’s title bar or tab.
  • Roman Urdu Description:
    <title> tag webpage ka title specify karta hai, jo browser ke title bar ya tab mein dikhayi deta hai.
  • Example Code:
    <title>My Awesome Webpage</title>

• What is <body> Element Tag?:

  • English Description:
    The <body> tag contains the main content of the webpage that is visible to the user.
  • Roman Urdu Description:
    <body> tag webpage ka main content rakhta hai jo user ko dikhayi deta hai.
  • Example Code:
    <body>
    <h1>Welcome to my website!</h1>
    <p>This is a paragraph inside the body tag.</p>
</body>

• What are <h1> to <h6> Element Tags?:

  • English Description:
    These tags represent different levels of headings, with <h1> being the most important and <h6> the least.
  • Roman Urdu Description:
    Yeh tags headings ke different levels ko represent karte hain, jisme <h1> sabse zaroori hota hai aur <h6> sabse kam.
  • Example Code:
    <h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Smaller Heading</h3>

• What is <p> Element Tag?:

  • English Description:
    The <p> tag is used to define paragraphs in HTML.
  • Roman Urdu Description:
    <p> tag HTML mein paragraphs ko define karne ke liye use hota hai.
  • Example Code:
    <p>This is a paragraph tag.</p>

• What is <a> Element Tag?

  • English Description:
    The <a> tag defines a hyperlink, allowing users to click and navigate to another page.
  • Roman Urdu Description:
    <a> tag ek hyperlink define karta hai, jo users ko click karke doosri page pe le jata hai.
  • Example Code:
    <a href="https://www.example.com">Visit Example</a>
