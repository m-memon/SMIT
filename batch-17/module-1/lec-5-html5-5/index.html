HTML5 List Structures

• There are Three Types of Lists:

  • English Description:
    In HTML, there are three main types of lists: Unordered lists (with bullet points), ordered lists (with numbers), and description lists (for defining terms).
  • Roman Urdu Description:
    HTML mein teen tareeqay ke lists hotay hain: Unordered lists (bullet points ke saath), ordered lists (numbers ke saath), aur description lists (terms ko define karne ke liye).
  • Example Code:
    <!-- Unordered List -->
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<!-- Ordered List -->
<ol>
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
</ol>

<!-- Description List -->
<dl>
  <dt>HTML</dt>
  <dd>Hypertext Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

• Type Attributes of Unordered List in HTML5:

  • English Description:
    The <ul> tag in HTML creates an unordered list. The type attribute can be used to specify the style of bullets, such as circles or squares.
  • Roman Urdu Description:
    HTML mein <ul> tag se unordered list banai jaati hai. type attribute ka istemal bullets ka style specify karne ke liye hota hai, jaise ke circles ya squares.
  • Example Code:
    <ul type="circle">
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

• Type Attributes of Ordered List in HTML5:

  • English Description:
    The <ol> tag is used for ordered lists, where list items are numbered. The type attribute can be used to change the numbering style, such as Roman numerals or letters.
  • Roman Urdu Description:
    HTML mein <ol> tag ordered list ke liye use hota hai, jahan list items numbers ke saath hotay hain. type attribute ka istemal numbering style ko change karne ke liye hota hai, jaise Roman numerals ya letters.
  • Example Code:
    <ol type="A">
  <li>First</li>
  <li>Second</li>
</ol>

• Do Nested Lists Work:

  • English Description:
    Yes, nested lists work in HTML. You can place one list inside another to create complex, hierarchical structures.
  • Roman Urdu Description:
    Haan, nested lists HTML mein kaam karti hain. Aap ek list ke andar doosri list daal kar complex aur hierarchical structures bana sakte hain.
  • Example Code:
    <ul>
  <li>Main Item 1
    <ul>
      <li>Sub Item A</li>
      <li>Sub Item B</li>
    </ul>
  </li>
  <li>Main Item 2</li>
</ul>

• Structure of Nested List:

  • English Description:
    A nested list is a list inside another list. It’s created by placing a list (<ul> or <ol>) inside a list item (<li>) of another list.
  • Roman Urdu Description:
    Nested list ek list hoti hai jo doosri list ke andar hoti hai. Isay ek list (<ul> ya <ol>) ko doosri list ke <li> item ke andar rakh kar banaya jaata hai.
  • Example Code:
    <ol>
  <li>Item 1
    <ul>
      <li>Sub-item A</li>
      <li>Sub-item B</li>
    </ul>
  </li>
  <li>Item 2</li>
</ol>

• <a> Element Tag in HTML5:

  • English Description:
    The <a> tag in HTML is used to create hyperlinks, which allow navigation to other pages, files, or resources.
  • Roman Urdu Description:
    HTML mein <a> tag hyperlinks banane ke liye use hota hai, jo aapko doosri pages, files, ya resources tak le jaate hain.
  • Example Code:
    <a href="https://www.example.com">Click here</a>

• href Attribute of <a> Element Tag in HTML5:

  • English Description:
    The href attribute in the <a> tag specifies the URL or path where the link points to. It stands for "Hypertext Reference."
  • Roman Urdu Description:
    href attribute <a> tag mein URL ya path specify karta hai jahan link point karta hai. Ye "Hypertext Reference" ka short form hai.
  • Example Code:
    <a href="https://www.example.com" target="_blank">Visit Example</a>
