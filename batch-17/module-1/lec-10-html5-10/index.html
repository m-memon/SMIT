Forms and Deployment

• What is Domain:

  • English Description:
    A domain is the human-readable address of a website, like example.com.
  • Roman Urdu Description:
    Domain website ka human-readable address hota hai, jaise example.com.
  • Example Code:
    *(Domains don’t use code, but for explanation:)*
https://www.example.com

• What is Hosting:

  • English Description:
    Hosting is a service that stores your website files and makes them accessible on the internet.
  • Roman Urdu Description:
    Hosting ek service hai jo website ke files ko store karti hai aur internet par access dene ke qabil banati hai.
  • Example Code:
    *(No code example; concept-based)*
Use platforms like: Hostinger, Bluehost, Netlify

• How to deploy a project on Netlify:

  • English Description:
    Deploy your project by uploading the build folder or connecting your GitHub repository to Netlify.
  • Roman Urdu Description:
    Apna project deploy karne ke liye build folder upload karein ya GitHub repository ko Netlify se connect karein.
  • Example Code:
    1. Create account on https://www.netlify.com  
2. Click "Add New Site" → Import from Git  
3. Select repository → Configure → Deploy

• How to deploy a project on GitHub:

  • English Description:
    Upload your project to a GitHub repository and enable GitHub Pages from repository settings.
  • Roman Urdu Description:
    Apne project ko GitHub par upload karein aur GitHub Pages ko settings se enable karein.
  • Example Code:
    1. git init  
2. git remote add origin <repo-url>  
3. git push -u origin main  
4. Settings → Pages → Select branch → Save

• <form> element tag in HTML5:

  • English Description:
    Defines a section of the document that contains interactive controls to submit user input.
  • Roman Urdu Description:
    Yeh document ka woh hissa define karta hai jahan user input dena aur submit karna mumkin hota hai.
  • Example Code:
    <form action="/submit">
    <input type="text" name="name" />
</form>

• <label> element tag of <form> in HTML5:

  • English Description:
    Defines a label for an input element, improving accessibility and form clarity.
  • Roman Urdu Description:
    Yeh input element ke liye label provide karta hai, jo accessibility aur clarity improve karta hai.
  • Example Code:
    <label for="email">Email:</label>
<input id="email" type="email" />

• <input /> element tag of <form> in HTML5:

  • English Description:
    Used to create various types of user input fields within a form.
  • Roman Urdu Description:
    Form ke andar mukhtalif qisam ke input fields banane ke liye use hota hai.
  • Example Code:
    <input type="text" name="username" />

• action attribute of <form> in HTML5:

  • English Description:
    Specifies where to send the form data when submitted.
  • Roman Urdu Description:
    Yeh batata hai ke form data submit hone ke baad kahan bhejna hai.
  • Example Code:
    <form action="/submit-data">

• type attribute of <input /> in <form>:

  • English Description:
    Specifies the type of input like text, password, email, etc.
  • Roman Urdu Description:
    Yeh input ka type define karta hai jaise text, password, ya email.
  • Example Code:
    <input type="password" />

• maxlength attribute of <input /> in <form>:

  • English Description:
    Limits the number of characters allowed in the input field.
  • Roman Urdu Description:
    Input field mein kitne characters allow hain, iski limit set karta hai.
  • Example Code:
    <input type="text" maxlength="10" />

• name attribute of <input /> in <form>:

  • English Description:
    Specifies the name of the input for identifying form data on submission.
  • Roman Urdu Description:
    Form submit hone par input ki pehchan ke liye iska naam set karta hai.
  • Example Code:
    <input name="username" />

• Group <input /> in <form>:

  • English Description:
    You can group related input fields like radio buttons using the same name.
  • Roman Urdu Description:
    Aap related input fields ko group kar sakte hain, jaise radio buttons mein same name dena.
  • Example Code:
    <input type="radio" name="gender" value="male" />
<input type="radio" name="gender" value="female" />

• required attribute of <input /> in <form>:

  • English Description:
    Forces the user to fill out the input field before submitting the form.
  • Roman Urdu Description:
    User ko form submit karne se pehle field ko fill karna zaroori banata hai.
  • Example Code:
    <input type="email" required />

• accept attribute of <input /> in <form>:

  • English Description:
    Specifies the types of files that the server accepts when file input is used.
  • Roman Urdu Description:
    Jab file upload ki jaye to server kaunse file types accept karega, yeh define karta hai.
  • Example Code:
    <input type="file" accept=".jpg, .png" />

• for attribute of <label> in <form>:

  • English Description:
    Links the label to a specific input field using its id.
  • Roman Urdu Description:
    Label ko kisi input field ke id se link karta hai.
  • Example Code:
    <label for="username">Username:</label>
<input id="username" type="text" />

• value attribute of <input /> in <form>:

  • English Description:
    Specifies the default value for the input field.
  • Roman Urdu Description:
    Input field ka default value set karta hai.
  • Example Code:
    <input type="text" value="John Doe" />
