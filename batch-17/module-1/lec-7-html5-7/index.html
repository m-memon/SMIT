HTML5 Links & Images

• index.html is used as the default file in HTML5:

  • English Description:
    The index.html file is the default or starting page of a website in HTML5. When users visit a website without specifying a page, the server automatically looks for this file and loads it.
  • Roman Urdu Description:
    index.html file website ka default ya starting page hota hai HTML5 mein. Jab users website pe bina kisi specific page ke visit karte hain, to server is file ko automatically search kar ke load karta hai.
  • Example Code:
    <!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Home Page</title>
</head>
<body>
    <h1>Welcome to Our Website!</h1>
</body>
</html>

• ../ syntax is used for linking in HTML5:

  • English Description:
    The ../ syntax is used to link to a directory above the current directory. It helps navigate the file structure of a website when linking to resources or files located in parent directories.
  • Roman Urdu Description:
    ../ syntax ko parent directory ya upar ke folder mein link karne ke liye use kiya jata hai. Yeh website ke file structure ko navigate karne mein madad karta hai jab resources ya files parent directories mein hoti hain.
  • Example Code:
    <a href="../about.html">About Us</a>

• <img> element tag in HTML5:

  • English Description:
    The <img> tag is used to embed images in a web page. It does not have a closing tag and requires attributes like src and alt.
  • Roman Urdu Description:
    The <img> tag is used to embed images in a web page. It does not have a closing tag and requires attributes like src and alt.
  • Example Code:
    <img src="image.jpg" alt="A beautiful scenery">

• src attribute of <img> element tag in HTML5:

  • English Description:
    The src attribute of the <img> tag specifies the path or URL of the image to be displayed.
  • Roman Urdu Description:
    <img> tag ka src attribute image ki location ya URL specify karta hai jo display karni hoti hai.
  • Example Code:
    <img src="image.jpg" alt="A beautiful beach">

• height attribute of <img> element tag in HTML5:

  • English Description:
    The height attribute is used to define the height of an image in the <img> tag. The value is specified in pixels or other units.
  • Roman Urdu Description:
    height attribute <img> tag mein image ki height ko define karta hai. Iska value pixels ya doosre units mein hota hai.
  • Example Code:
    <img src="image.jpg" alt="Image" height="200">

• width attribute of <img> element tag in HTML5:

  • English Description:
    The width attribute is used to define the width of an image in the <img> tag. Like the height attribute, the value can be in pixels or other units.
  • Roman Urdu Description:
    width attribute <img> tag mein image ki width ko define karta hai. Jaise height attribute, iska value bhi pixels ya doosre units mein hota hai.
  • Example Code:
    <img src="image.jpg" alt="Image" width="300">

• title attribute of <img> element tag in HTML5:

  • English Description:
    The title attribute provides additional information about the image. It appears when the user hovers over the image.
  • Roman Urdu Description:
    title attribute image ke baare mein additional information provide karta hai. Yeh tab appear hota hai jab user image par hover karta hai.
  • Example Code:
    <img src="image.jpg" alt="Image" title="This is a beautiful landscape">

• alt attribute of <img> element tag in HTML5:

  • English Description:
    The alt> attribute describes the content of an image. It is displayed if the image cannot be loaded or for accessibility purposes (screen readers).
  • Roman Urdu Description:
    alt attribute image ke content ko describe karta hai. Agar image load nahi hotib> ya accessibility ke liye (screen readers) yeh text display hota hai.
  • Example Code:
    <img src="image.jpg" alt="A beautiful sunset over the ocean">

• <figure> element tag of HTML5:

  • English Description:
    The <figure> tag is used to group content like images, diagrams, or illustrations along with their captions.
  • Roman Urdu Description:
    <figure> tag ko images, diagrams ya illustrations ko unki captions ke sath group karne ke liye use kiya jata hai.
  • Example Code:
    <figure>
    <img src="image.jpg" alt="Scenery">
    <figcaption>Beautiful view of the mountains</figcaption>
</figure>

• <figcaption> element tag in <figure> element tag of HTML5:

  • English Description:
    The <figcaption> tag is used to provide a caption or description for the content within a <figure> element.
  • Roman Urdu Description:
    <figcaption> tag <figure> element ke andar content ke liye caption ya description dene ke liye use hota hai.
  • Example Code:
    <figure>
    <img src="image.jpg" alt="Image of a park">
    <figcaption>This is a beautiful park in the city</figcaption>
</figure>
