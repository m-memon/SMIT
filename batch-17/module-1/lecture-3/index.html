HTML5 Elements Overview

• Shift + 1: HTML5 Structure Shortcut:

  • English Description:
    This shortcut generates a basic HTML5 boilerplate structure, including <!DOCTYPE html>, <html>, <head>, and <body> tags.
  • Roman Urdu Description:
    Yeh shortcut ek bunyadi HTML5 structure generate karta hai, jis mein <!DOCTYPE html>, <html>, <head>, aur <body> tags shamil hotay hain.
  • Example Code:
    <!DOCTYPE html>
    <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
        </head>
        <body>
            <!-- Content goes here -->
        </body>
</html>

• <sup> Element Tag:

  • English Description:
    The <sup> tag is used to display text as superscript, commonly for exponents or ordinal numbers.
  • Roman Urdu Description:
    <sup> tag ka istemal text ko superscript ke tor par dikhane ke liye hota hai, jo aam tor par exponents ya ordinal numbers ke liye use hota hai.
  • Example Code:
    <p>E = mc<sup>2</sup></p>

• <sub> Element Tag:

  • English Description:
    The <sub> tag is used to display text as subscript, often for chemical formulas or footnotes.
  • Roman Urdu Description:
    <sub>
    b> tag ka istemal text ko subscript ke tor par dikhane ke liye hota hai, jo aam tor par chemical formulas ya footnotes ke liye use hota hai.
  • Example Code:
    <p>H<sub>2</sub>O</p>

• HTML5 Ignores Extra Spaces:

  • English Description:
    HTML5 collapses multiple consecutive spaces into a single space when rendering content.
  • Roman Urdu Description:
    HTML5 content render karte waqt mutadid consecutive spaces ko ek single space mein tabdeel kar deta hai.
  • Example Code:
    <p>This    is    spaced    text.</p>

• <hr /> Element Tag:

  • English Description:
    The <hr /> tag represents a thematic break or horizontal rule, often used to separate content sections.
  • Roman Urdu Description:
    <hr /> tag ek thematic break ya horizontal rule ko represent karta hai, jo aam tor par content sections ko separate karne ke liye use hota hai.
  • Example Code:
    <hr />

• <br /> Element Tag:

  • English Description:
    The <br /> tag inserts a line break within text, useful for poems or addresses.
  • Roman Urdu Description:
    <br /> tag text ke andar line break insert karta hai, jo poems ya addresses ke liye mufeed hota hai.
  • Example Code:
    <p>
    Line one.
    <br />
    Line two.
</p>

• Semantic Element Tags in HTML5:

  • English Description:
    Semantic elements like <article>, <section>, and <nav> provide meaning to the content, improving accessibility and SEO.
  • Roman Urdu Description:
    Semantic elements jaise <article>, <section>, aur <nav> content ko meaning dete hain, jo accessibility aur SEO ko behtar banata hai.
  • Example Code:
    <article>
    <h2>Article Title</h2>
    <p>Content of the article.</p>
</article>

• <strong> Element Tag:

  • English Description:
    The <strong> tag indicates that its contents have strong importance, typically rendered in bold.
  • Roman Urdu Description:
    <strong> tag is baat ko indicate karta hai ke iske andar ka content bohat zaroori hai, jo aam tor par bold mein render hota hai.
  • Example Code:
    <p>
    <strong>
        Warning:
    </strong> This is important.
</p>

• <em> Element Tag:

  • English Description:
    The <em> tag represents text with emphasis, usually rendered in italics.
  • Roman Urdu Description:
    <em>; tag text ko emphasis dene ke liye use hota hai, jo aam tor par italics mein render hota hai.
  • Example Code:
    <p>
    <em>
        Note:
    </em> This is emphasized text.
</p>

• <blockquote> Tag:

  • English Description:
    The <blockquote> tag is used for extended quotations, typically indented.
  • Roman Urdu Description:
    <blockquote> tag lambi quotations ke liye use hota hai, jo aam tor par indented hoti hain.
  • Example Code:
    <blockquote cite="https://example.com">
    <p>Extended quotation text.</p>
</blockquote>

• <q> Element Tag:

  • English Description:
    The <q> tag represents inline quotations, usually enclosed in quotation marks.
  • Roman Urdu Description:
    <q> tag inline quotations ke liye use hota hai, jo aam tor par quotation marks mein enclosed hota hai.
  • Example Code:
    <p>
    <q>
        Inline quoted text.
    </q>
</p>

• cite Attribute of <blockquote> Element Tag in HTML5:

  • English Description:
    Specifies the source URL of the quoted content.
  • Roman Urdu Description:
    Quoted content ka source URL specify karta hai.
  • Example Code:
    <blockquote cite="https://www.example.com/article">
    This is a quoted paragraph.
</blockquote>

• <abbr> Element Tag in HTML5:

  • English Description:
    Defines an abbreviation or acronym.
  • Roman Urdu Description:
    Kisi mukhtasir lafz ya acronym ko define karta hai.
  • Example Code:
    <p>
    <abbr title="World Health Organization">
        WHO
    </abbr> is an agency of the United Nations.
</p>

• title Attribute of <abbr> Element Tag in HTML5:

  • English Description:
    Provides the full form or explanation of the abbreviation when hovered.
  • Roman Urdu Description:
    Mouse le jaane par abbreviation ka full form ya tafseel dikhata hai.
  • Example Code:

• <cite> Element Tag in HTML5:

  • English Description:

  • Roman Urdu Description:

  • Example Code:
    <p>
    <cite>
        The Great Gatsby
    </cite> is a novel by F. Scott Fitzgerald.
</p>

• <s> Element Tag in HTML5:

  • English Description:
    Shows text that is no longer accurate or relevant, typically rendered with a strikethrough.
  • Roman Urdu Description:
    Aisa text dikhata hai jo ab ghalat ya purana ho, usually strikethrough ke sath.
  • Example Code:
    <p>
    The old price was
    <s>
        $50
    </s>, now it's $30.
</p>
