Form Element Basics

• <select> element tag of <form> element tag in HTML5:

  • English Description:
    The <select> tag is used to create a dropdown list of options within a form. It allows users to choose one or more options from the list.
  • Roman Urdu Description:
    <select> tag form mein ek dropdown list banata hai jisme users ko multiple ya ek option choose karne ki sahulat milti hai.
  • Example Code:
    <form>
    <label for="cars">
        Choose a car:
    </label>
    <select id="cars" name="cars">
        <option value="volvo">
            Volvo
        </option>
        <option value="saab">
            Saab
        </option>
        <option value="mercedes">
            Mercedes
        </option>
        <option value="audi">
            Audi
        </option>
    </select>
</form>

• name attribute in <select> element tag of <form> element tag in HTML5:

  • English Description:
    The name attribute specifies the name for the dropdown. It is used to reference form data after submitting the form.
  • Roman Urdu Description:
    name attribute dropdown ki ek pehchaan (identifier) deta hai, jise form submit hone par data reference ke liye use kiya jata hai.
  • Example Code:
    <form>
    <select name="carModel">
        <option value="volvo">
            Volvo
        </option>
        <option value="saab">
            Saab
        </option>
        <option value="mercedes">
            Mercedes
        </option>
    </select>
</form>

• <option> element tag in <select> element tag of <form> in HTML5:

  • English Description:
    The <option> tag is used to define options inside the <select> dropdown. Each <option> element defines a single option for the user to select.
  • Roman Urdu Description:
    <option> tag <select> dropdown ke andar alag-alag options define karta hai, jisme har <option> ek option ko represent karta hai jo user choose kar sakta hai.
  • Example Code:
    <select name="fruit">
    <option value="apple">
        Apple
    </option>
    <option value="banana">
        Banana
    </option>
    <option value="cherry">
        Cherry
    </option>
</select>

• method attribute of <form> element tag in HTML5:

  • English Description:
    The method attribute specifies how to send form data. The two common methods are GET (appends data to the URL) and POST (sends data as the body of the request).
  • Roman Urdu Description:
    method attribute yeh batata hai ke form ka data kaise bhejna hai. Do aam methods hain: GET (jo data ko URL mein append karta hai) aur POST (jo data ko request ke body mein bhejta hai).
  • Example Code:
    <form action="/submit" method="POST">
    <input type="text" name="username">
    <input type="submit" value="Submit">
</form>

• placeholder attribute in <input /> element tag of <form> element tag in HTML5:

  • English Description:
    The placeholder attribute provides a hint or example to the user about what to enter in the input field.
  • Roman Urdu Description:
    placeholder attribute user ko input field mein kya dalna hai, iski ek misaal ya hint deta hai.
  • Example Code:
    <form>
    <input type="text" placeholder="Enter your name">
</form>

• <center> element tag in HTML5:

  • English Description:
    The <center> tag is used to center-align content. However, it is now considered obsolete in HTML5 and CSS is recommended for centering content.
  • Roman Urdu Description:
    <center> tag content ko center mein align karta tha, lekin HTML5 mein yeh tag obsolete ho chuka hai, aur ab content ko center karne ke liye CSS use karna chahiye.
  • Example Code:
    <center>
    <h1>
        This is centered text
    </h1>
</center>

• <fieldset> element tag in HTML5:

  • English Description:
    The <fieldset> tag is used to group related elements within a form. It is often used to visually group sections of a form.
  • Roman Urdu Description:
    <fieldset> tag form ke andar related elements ko group karne ke liye use hota hai. Yeh aksar form ke sections ko visually group karne ke liye istemal hota hai.
  • Example Code:
    <form>
    <fieldset>
        <legend>
            Personal Information
        </legend>
        Name: <input type="text">
        Age: <input type="number">
    </fieldset>
</form>

• <legend> element tag of <fieldset> element tag in HTML5:

  • English Description:
    The <legend> tag is used to provide a caption or title for the <fieldset> element, helping to explain the content inside the group.
  • Roman Urdu Description:
    <legend> tag <fieldset> element ke liye ek title ya caption provide karta hai, jo andar ke content ko explain karta hai.
  • Example Code:
    <form>
    <fieldset>
        <legend>
            Contact Information
        </legend>
        Email: <input type="email">
        Phone: <input type="tel">
    </fieldset>
</form>
