HTML5 Elements and Functionality

• class attribute in HTML5:

  • English Description:
    The class attribute is used to assign one or more class names to an HTML element, allowing you to apply CSS styles and JavaScript functions to multiple elements at once.
  • Roman Urdu Description:
    class attribute HTML element ko ek ya zyada class names assign karta hai, jisse aap CSS styles aur JavaScript functions ko multiple elements par ek saath apply kar sakte hain.
  • Example Code:
    <div class="box">This is a div with a class.</div>

• id attribute in HTML5:

  • English Description:
    The id attribute assigns a unique identifier to an HTML element, which must be unique within the document.
  • Roman Urdu Description:
    id attribute ek unique identifier HTML element ko assign karta hai, jo document mein unique hona chahiye.
  • Example Code:
    <div id="uniqueElement">This is a div with an ID.</div>

• Difference between class & id attribute in HTML5:

  • English Description:
    The class attribute can be used on multiple elements, whereas the id attribute is used for one unique element in the document.
  • Roman Urdu Description:
    class attribute ko aap multiple elements par use kar sakte hain, jabke id attribute sirf ek unique element ke liye use hota hai.
  • Example Code:
    <div class="box">This is a div with a class.</div>
<div id="uniqueElement">This is a div with an ID.</div>

• <div> element tag in HTML5:

  • Roman Urdu Description:
    <div> tag ek block-level element hai jo content ko group karne ke liye styling ya scripting ke liye use hota hai.
  • Example Code:

• <span> element tag in HTML5:

  • English Description:
    The <span> tag is an inline container used to apply styles or scripts to small portions of text or other inline elements.
  • Roman Urdu Description:
    <span> tag ek inline container hai jo chhoti portions of text ya inline elements ko style ya script apply karne ke liye use hota hai.

• <section> element tag in HTML5:

  • English Description:
    The <section> tag is used to define a thematic section of content in an HTML document.
  • Roman Urdu Description:
    <section> tag HTML document mein content ke ek thematic section ko define karta hai.
  • Example Code:
    <section>
    <h2>Section Title</h2>
    <p>This is some content in the section.</p>
</section>

• <header> element tag in HTML5:

  • English Description:
    The <header> tag is used to define introductory content or a set of navigational links, typically at the top of the page or section.
  • Roman Urdu Description:
    <header> tag introductory content ya navigation links ka group define karta hai, jo aksar page ya section ke upar hota hai.
  • Example Code:
    <header>
    <h1>Page Header</h1>
    <nav>Navigation Links</nav>
</header>

• <nav> element tag of <header> element tag in HTML5:

  • English Description:
    The <nav> element is used to define navigation links within a webpage, often placed inside a <header> element to group the site's navigation.
  • Roman Urdu Description:
    <nav> element webpage ke andar navigation links define karne ke liye use hota hai, jo aksar <header> element ke andar rakha jata hai taake site ka navigation group ho sake.
  • Example Code:
    <header>
    <nav>
        <ul>
            <li><a href="#home">Home</a></li>
            <li><a href="#about">About</a></li>
            <li><a href="#services">Services</a></li>
            <li><a href="#contact">Contact</a></li>
        </ul>
    </nav>
</header>

• poster attribute of <video> element tag in HTML5:

  • English Description:
    The poster attribute specifies an image to show before the video starts playing, usually used as a preview thumbnail.
  • Roman Urdu Description:

  • Example Code:

    <source src="video.mp4" type="video/mp4">
</video>
