HTML5 Linking & Elements

• There are Two Types of Linking in HTML5:

  • English Description:
    In HTML5, linking can be done in two main ways: relative linking and absolute linking. These methods are used to link web pages, files, and resources.
  • Roman Urdu Description:
    HTML5 mein linking do tareeqon se ki ja sakti hai: relative linking aur absolute linking. In tareeqon ka use web pages, files, aur resources ko link karne ke liye hota hai.
  • Example Code:
    <!-- Relative Linking -->
<a href="contact.html">Contact Us</a>

<!-- Absolute Linking -->
<a href="https://www.example.com">Visit Example</a>

• Relative Linking in HTML5:

  • English Description:
    Relative linking refers to linking files based on the current file's directory. It doesn’t require the full URL, only the path relative to the current file.
  • Roman Urdu Description:
    Relative linking ka matlab hai files ko current file ke directory ke hisaab se link karna. Ismein poora URL ki zaroorat nahi hoti, sirf current file ke hisaab se path diya jata hai.
  • Example Code:
    <a href="about.html">About Us</a>

• Absolute Linking in HTML5:

  • English Description:
    Absolute linking uses the full URL (including the protocol like HTTP/HTTPS) to link to external websites or resources.
  • Roman Urdu Description:
    Absolute linking mein poora URL use hota hai, jo ke external websites ya resources ko link karta hai. Ismein protocol (jaise HTTP ya HTTPS) bhi shamil hota hai.
  • Example Code:
    <a href="https://www.example.com">Visit Example</a>

• There are Two Types of Elements in HTML5:

  • English Description:
    In HTML5, elements are categorized into two types: block-level elements and inline elements. Block-level elements start on a new line, while inline elements do not.
  • Roman Urdu Description:
    HTML5 mein elements ko do qisam mein divide kiya jata hai: block-level elements aur inline elements. Block-level elements nayi line se shuru hote hain, jabke inline elements same line mein rehte hain.
  • Example Code:
    <!-- Block-level element -->
<div>This is a block-level element</div>

<!-- Inline element -->
<span>This is an inline element</span>

• Block-Level Elements in HTML5:

  • English Description:
    Block-level elements in HTML5 occupy the full width of their container and start on a new line. Examples include <div>, <p>, and <h1>.
  • Roman Urdu Description:
    Block-level elements HTML5 mein apne container ki puri width lete hain aur naye line par start hote hain. Iske examples mein <div>, <p>, aur <h1> shamil hain.
  • Example Code:
    <div>This is a block-level element</div>
<p>This is a paragraph, which is also a block-level element.</p>

• Inline Elements in HTML5:

  • English Description:
    Inline elements only take up as much width as needed and do not cause a new line. Examples include <span>, <a>, and <strong>.
  • Roman Urdu Description:
    Inline elements HTML5 mein sirf utni width lete hain jitni unhe zaroorat hoti hai aur nayi line create nahi karte. Iske examples mein <span>, <a>, aur <strong> aate hain.
  • Example Code:
    <span>This is an inline element</span>
<a href="https://www.example.com">Click Here</a>

• target Attribute of <a> Element Tag in HTML5:

  • English Description:
    The target attribute in the <a> tag specifies where to open the linked document. Common values include _blank (opens in a new window) and _self (opens in the same window).
  • Roman Urdu Description:
    target attribute <a> tag mein specify karta hai ke linked document kaha open hoga. Aam values mein _blank (naye window mein open hota hai) aur _self (same window mein open hota hai) shamil hain.
  • Example Code:
    <a href="https://www.example.com" target="_blank">Visit Example</a>

• ./ Syntax Used for Linking in HTML5:

  • English Description:
    The ./ syntax in HTML5 is used to link to a file in the current directory. It indicates a relative path from the current location.
  • Roman Urdu Description:
    HTML5 mein ./ syntax current directory mein maujood file ko link karne ke liye use hota hai. Ye current location se relative path indicate karta hai.
  • Example Code:
    <a href="./about.html">About Us</a>
